# Granny square glossary

This documentation uses US crochet terminology.

| Term | Abbreviation | Description |
| :---- | :---- | :---- |
| **Chain stitch** | ch | A small freestanding stitch, usually used to create  empty **spaces**, or to connect larger stitches. |
| **Chain-2 space** | ch2 sp | The empty space under a set of two **chain stitches**. |
| **Double crochet** | dc | The primary crochet stitch in a granny square. Approximately the same height as three **chain stitches**. |
| **Granny cluster** | \- | A group of three **double crochet** stitches. |
| **In the round** | \- | A style of crochet based on starting in the centre and working outward in a spiral. See Introduction to granny squares for details. |
| **Slip stitch** | sl st | A small connecting stitch, usually used to connect yarn to a hook or to a project. Also commonly used to complete a row in projects made **in the round**.  |
| **Space** | sp | An empty space in a row of crochet, usually appearing underneath **chain stitches** or between **granny clusters**. |
